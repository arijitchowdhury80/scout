# Scout docs (Mintlify)

Preview locally with `npx mint dev` from this directory (Mintlify's current CLI is published as the `mint` package; if that command has been renamed by the time you read this, check `npx mintlify@latest --help` as a fallback).

In the Mintlify dashboard, point the project at the `docs-site/` path of this repo (not the repo root), and set the custom domain to `docs.scout.chowmes.com`.
