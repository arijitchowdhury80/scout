Features

Acquisition primitives for real workflows.

Features in Scout are organized by what people need to do: acquire pages, preserve proof, create typed records, and export something downstream tools can trust.

Product catalogs

Extract listing records, preserve source evidence, and prepare exports for Algolia, JSONL, CSV, SQLite, or spreadsheet adapters.

Competitive intelligence

Collect company, investor, careers, news, docs, and website-quality evidence into repeatable artifact folders.

Agent tools

Give local agents a controlled acquisition layer with provenance instead of raw, untraceable fetches.

Demo and search builds

Turn a prospect or customer website into inspectable product records that can feed Algolia demos or evaluation indexes.

Research archives

Save the raw evidence, normalized records, and extraction report so future runs can compare what changed.

Browser-assisted capture

Use captured DOM and screenshots when normal crawlers are blocked, without pretending every site can be bypassed.
